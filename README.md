## Задание 1 
Формулы для решения задач:
* x < 0 : y = -0.5*x
* 0 <= x <= 2 : x^2 + (y - 2)^2 = 4 {y <= 2}
* 2 < x < 4 : (x - 2)^2 + y^2 = 4 {y >= 0}
* x >= 4 : y = -(x - 4)/2
